# Starfield NG

 - C++ reincarnation of the Windows Starfield Simulation screensaver.

Written in raw Win32, this fork is modernized, with new/expanded options added.

## Motivation

I always wanted to be able to add more/less stars than the original one allowed, and to adjust the color.
Plus, they stopped shipping this screensaver after Windows XP.

## Building
You can use the Makfile, or GN/Ninja with the BUILD.gn file. Both use MinGW as their toolchain.
